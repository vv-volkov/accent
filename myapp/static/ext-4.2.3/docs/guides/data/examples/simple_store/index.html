Simple Store

This example creates a simple store that auto-loads its data from an AJAX proxy. Since we are only dealing with data there is no UI, so a global variable called "userStore" is created which is an instance of Ext.data.Store.

Feel free to experiment with the "userStore" object on the console command line. For example - 'userStore.getCount()' gets the total number of records in the store. 'userStore.getAt(0)'' gets the record at index 0.

See console for output.
